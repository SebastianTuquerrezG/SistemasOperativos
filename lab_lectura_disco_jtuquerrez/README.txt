Deberá desarrollar un programa en C, que tome como entrada el primer sector de un disco duro
inicializado como “MBR” y con una tabla de particiones válida, e imprimir la información de la
tabla de particiones.

Para la ejecucion de este programa, debera compilar con 
    make

Ejecutar el programa mediante
    .hdisk archivo_mbr
    
    Este archivo mbr sera el primer sector del disco duro que leera el programa
